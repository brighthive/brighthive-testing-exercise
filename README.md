# BrightHive QA Automation Exercise

This repository contains a Senior QA Automation Engineer technical exercise.

## Exercise Overview

Candidates are asked to:
1. Build an automated test framework (focusing on Swagger UI automation)
2. Find and document bugs in a web application
3. Create comprehensive bug reports

## Quick Start

```bash
cd senior_qa_exercise
./start.sh
```

The webapp will be running at **http://localhost:8000**

**Swagger UI**: http://localhost:8000/docs
